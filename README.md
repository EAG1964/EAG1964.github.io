# 🔬 PhyChem Virtual Lab | EAG1964

**PhyChem Virtual Lab** es una plataforma de recursos interactivos de **Física y Química** diseñada para Secundaria (ESO) y Bachillerato.

## 🌐 Acceso a la Web
Puedes explorar el laboratorio en vivo aquí:

---

## 📚 Contenido Actual
El laboratorio organiza las simulaciones por niveles educativos:

* **2º ESO:** Cambio de unidades y Modelos atómicos.
* **3º y 4º ESO:** Estructura de la materia.
* **1º Bachillerato:** Espectroscopía.
* **2º Bachillerato:** Física Cuántica (Física) y Estructura Atómica (Química).
